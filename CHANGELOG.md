### バグ修正

- ログの設定情報出力が1.1.0時点の項目のままだった問題を修正

### バグ修正

- 自動更新が正しく動作しない問題を修正

## 1.3.0 - 2025-11-30

### 改善

- 権限ごとの制限を追加: 管理者は設定変更+実行、編集者は実行のみに制限
- UI のスマートフォン・タブレットへの対応
- パフォーマンスの高速化
- 出力ファイルの最適化

### バグ修正

- GitLab へのプッシュ時に差分検出が機能していなかった問題を修正
- 設定リセット時にトークン（GitHub/GitLab/Cloudflare）がクリアできていなかった問題を修正

---

## 1.2.0 - 2025-11-29

### 機能追加

- GitLab 出力に対応
- Cloudflare Workers 出力に対応
- プラグインの自動更新に対応
- デバッグモードを追加
  （URL に `&debugmode=on` / `&debugmode=off` で切り替え）

### 改善

- UI の一部整理
- 未使用の CSS/JS/メディアファイルを出力から除外
- ZIP 圧縮率の最適化
- ログの簡略化

### セキュリティ

- 全体的なセキュリティー強化

---

## 1.1.0 - 2025-11-26

初回公開バージョン。

### パフォーマンス改善

- 差分検出機能により 2 回目以降の変換速度を改善
- バッチアップロード処理の実装
- メモリ効率の改善
- レート制限機能の追加
- 全体的なセキュリティの強化

### 機能追加

- タグ・日付・著者アーカイブの個別制御
- XML サイトマップ・RSS フィードの生成制御
- 処理の安全な中止機能
- 詳細な実行時間表示
- ローカル Git 出力を GitHub 設定から分離し、独立した出力先として追加

### UI 改善

- デフォルト設定でキャッシュと並列処理を有効化
- ログ表示の簡略化

### バグ修正

- GitHub 出力時の進捗表示が不正確だった問題を修正
- コミットメッセージのタイムスタンプが更新されない問題を修正
- 空のリポジトリへの初回コミット時にエラーで失敗する問題を修正
- ZIP 出力時に一時フォルダの構造が含まれる問題を修正

---
